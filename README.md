# Getting start

### Copy repo
`git clone https://github.com/DeltaDevSys/dotfiles`

### moving to ~/.config/
`mv ~/dotfiles ~/.config/`

### loading depends
``` fish
chmod +x ~/.config/scripts/load_depends.sh
~/.config/scripts/load_depends.sh
```
